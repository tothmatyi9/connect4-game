# Connect-4 Game

Ez a Connect-4 játék parancssoros változata Java-ban, Maven segítségével.

## Használat

A projekt Maven-t használ, tehát először telepítened kell a [Maven](https://maven.apache.org/) eszközt.

1. Töltsd le a projektet a GitHub-ról.
2. Navigálj a projekt mappájába.
3. Futtasd a következő parancsot a projekt felépítéséhez:
   ```bash
   mvn clean install
